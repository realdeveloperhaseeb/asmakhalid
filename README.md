# Asma Khalid — Portfolio

A single-page portfolio for **Asma Khalid**, LinkedIn Growth Expert.

Built with plain **HTML, CSS & JavaScript** — no frameworks, no build step. Uses the **Outfit** font and a black + violet brand palette drawn from Asma's own content.

## Features

- Animated gradient-glow background & film grain
- Scroll-reveal animations (IntersectionObserver)
- Animated stat counters
- 3D tilt cards, custom cursor glow & scroll progress bar
- Sticky glass navbar with active-section highlighting
- Real LinkedIn post showcase, experience timeline & client reviews
- Fully responsive + reduced-motion support

## Structure

```
index.html      # markup
styles.css      # styling & responsive rules
script.js       # interactions & animations
assets/         # images
```

## Run locally

Just open `index.html` in a browser, or serve the folder:

```bash
npx serve .
```

## Deploy (Vercel)

This is a static site — no configuration needed.

```bash
vercel
```

Or connect the GitHub repo `realdeveloperhaseeb/asmakhalid` to Vercel and deploy. The root directory is the project root; framework preset = **Other**.
